Directives

Directives are defined as classes that can add new behavior to the elements in the template or modify existing behavior.

For eg. ngFor, ngIf,SwitchCases etc.

How to use ?

  1. ng g directive directive-name
  2. Import this directive to app.module.ts
  3. Use directive selector in html file to apply
  4. In directive file import ElementRef
  5. Create instance of it in constructor
  6. Now apply custom styling to the element
  7. Check custom-style.directive.ts file for refrence
