Services

To pass the data in components we donot pass directly, instead we create a service and pass the data through it. SO that we can pass same data to many components.

Services are free from module and component.

How to use it ?

  • ng g service service-name
  • We create function to provide data in service file.
  • Now import the service in component.ts file
  • Now inject it in the constructor and then we can use the data
